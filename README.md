# UserVault Pro

A production-ready User Management REST API built with FastAPI and PostgreSQL.

## Tech Stack

- Python 3.11
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT Authentication
- pytest
- Docker (coming soon)

## Features

- User Registration
- User Login
- JWT Authentication
- User CRUD
- Logging
- Custom Exceptions
- API Documentation
